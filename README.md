Sistema de Requisições de Compras – Madville

Plataforma web desenvolvida para organizar e acelerar todo o fluxo de compras da Madville.
O sistema permite que colaboradores enviem solicitações por um formulário público, gerando protocolo automático e registro imediato no banco. No painel administrativo, administradores e compradores conseguem visualizar, filtrar, analisar, aprovar, rejeitar e finalizar requisições, com histórico completo e notificações ao solicitante.
Tudo funciona de forma centralizada, segura e responsiva, garantindo controle, rastreabilidade e eficiência no processo de compras.
